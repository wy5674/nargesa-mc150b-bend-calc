# nargesa-mc150b-bend-calc
Just a simple program that helps set the depth on a nargesa mc150b.
Inputs are are in feet and inchs formated like 5'11"3/4 or just inches like 4"1/2.
Output is in centimeters to match whats on the machine.
Width : is the straight measurement of how far apart the legs are
Higth : is going from the center of your width straight up to measure how tall it is
Make sure to set your zero on the machine and set it to the program gives you.
The bend is usally not enough and needs to be sent through a second time with slight adjustment due to compression and spring back of the metal.
